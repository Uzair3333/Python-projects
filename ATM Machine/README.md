# 🏦 Simple ATM Machine

A text-based ATM simulator built in Python that lets users check balance, deposit money, withdraw funds, and exit — all from the terminal.

---

## 🚀 Features
- Check account balance
- Deposit money
- Withdraw money (with validation)
- Exit the program gracefully
- Handles invalid inputs safely

---

## 🧠 Concepts Used
- Loops (`while`)
- Conditional statements (`if-elif-else`)
- Input validation using `try-except`
- Basic arithmetic operations

---

## ▶️ How to Run
```bash
python main.py
