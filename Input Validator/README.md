# 🧮 User Input Validation Tool

A simple and interactive **Python CLI validation tool** that checks the validity of:
- 👤 Usernames  
- 📧 Email addresses  
- 🎂 Ages  

Designed to demonstrate **string manipulation**, **input validation**, and **error handling** in Python — all while providing a clean and user-friendly command-line interface.

---

## 🚀 Features

- ✅ **Username Validation** — Ensures names contain only alphabets and spaces  
- ✅ **Email Validation** — Checks for structure, symbols, and domain formatting  
- ✅ **Age Validation** — Validates numerical input within a realistic range (0–120)  
- ✅ **Graceful Error Handling** — Handles invalid input with helpful messages  
- ✅ **Attractive CLI Output** — Neatly formatted and emoji-enhanced feedback
